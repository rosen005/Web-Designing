# WoodsHub

**WoodsHub** is a sleek, user-friendly furniture e-commerce website designed to showcase and sell a curated range of chairs and wooden furniture. The site combines clean design, responsive layout, and basic interactivity to provide a smooth browsing experience for users.

## 🌐 Live Preview

woodshub/
│
├── woodshub.html # Main HTML file
├── woodshub.css # Styling and layout
├── woodshub.js # JS for interactivity (e.g., menu toggle)
├── /img/ # Product and brand images
├── /box1.html - box12.html # Individual product detail pages
├── Admin Login.html # Admin login page
├── user login.html # User login page


## ✨ Features

- Responsive navigation menu
- Homepage with hero section
- Interactive product showcase (New Arrivals & Best Sellers)
- Product rating visuals with stars
- Admin and user login pages
- Partner brand section
- Newsletter subscription prompt
- Informative about section
- Contact info and footer

## 🛠️ Technologies Used

- **HTML5** – Structure
- **CSS3** – Styling and layout (with Boxicons)
- **JavaScript** – Basic interactivity
- **Boxicons** – Icon library
- **Responsive Design** – Optimized for multiple devices

## 🚀 How to Run

1. Clone the repository or download the ZIP.
2. Ensure all asset folders (`img/`, `.css`, `.js`) are in place.
3. Open `woodshub.html` in any modern web browser.
